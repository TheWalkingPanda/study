# 基本语法

### 1. 斜体和粗体
_斜体_
*斜体2*
**粗体**

### 2. 分级标题
```
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题
```

### 3. 外链接
[外链接](http://www.baidu.com)

### 4. 无序列表
* 无序列表1
+ 无序列表2
- 无序列表3

### 5. 有序列表
1. 有序列表1
2. 有序列表2
3. 有序列表3

### 6. 文字引用
> 我是文字引用
markdown换行：两个空格加换行

取消markdown换行：再次两个空格加换行

### 7. 行内代码块
`I'm code`

### 8. 代码块
```
代码块
```
